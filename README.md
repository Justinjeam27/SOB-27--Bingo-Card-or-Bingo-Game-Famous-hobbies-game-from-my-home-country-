# SOB-27--Bingo-Card-or-Bingo-Game-Famous-hobbies-game-from-my-home-country-
Bingo is a game of chance where players match numbers or symbols on a   (American, 75-ball) or   (British/European, 90-ball) card to randomly drawn, announced items. Players mark their cards, often using a "free space," and aim to complete a specific pattern (line, diagonal, or blackout) to shout "Bingo!" and win
